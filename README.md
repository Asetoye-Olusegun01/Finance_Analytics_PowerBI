# Finance Performance Dashboard (Power BI)

This project presents a comprehensive Finance Performance Dashboard built in Power BI. The dashboard visualizes revenue, expenses, profit margins, and monthly financial trends to help organizations monitor and optimize overall financial health.

## Key Insights
- Revenue and expenses show clear monthly patterns, highlighting seasonal financial cycles.
- Profit margins fluctuate across months, revealing periods of high and low profitability.
- Certain product categories contribute significantly more to overall revenue.
- High-expense areas can be easily identified for cost-optimization strategies.
- Consistent upward or downward financial trends are visible through year-over-year comparisons.

## Objectives of the Dashboard
- Track revenue, expenses, and profit over time.
- Calculate and monitor profit margin percentage.
- Identify high-performing and low-performing financial periods.
- Compare financial metrics across categories or departments.
- Enable teams to make informed budgeting and forecasting decisions.

## Dataset Description
The dataset includes the following fields:
- Transaction ID  
- Date  
- Product/Category  
- Region  
- Revenue  
- Expenses  
- Profit (calculated)  
- Profit Margin % (calculated)

Data cleansing and calculations were performed in Power Query, and advanced KPIs were created using DAX.

## Tools Used
- Power BI Desktop  
- Power Query Editor  
- DAX  
- Microsoft Excel
